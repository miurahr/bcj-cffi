Security Policy
===============

Supported Versions
------------------

Only recent development version are currently being supported with security updates.

+---------+--------------------+
| Version | Supported          |
+=========+====================+
| 0.5.x   | Development        |
+---------+--------------------+
| < 0.5   | not supported      |
+---------+--------------------+

Reporting a Vulnerability
-------------------------

Please disclose security vulnerabilities privately at miurahr@linux.com
